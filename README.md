# Tree Vertical Order Traversal Example

This Python example demonstrates the Vertical Order Traversal algorithm for binary trees. It uses a Breadth-First Search (BFS) approach to assign horizontal distances and levels to each node, then groups and sorts nodes to display them column by column. The output shows nodes ordered vertically from left to right, and top to bottom within each vertical column.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
